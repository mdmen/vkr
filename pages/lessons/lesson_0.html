(фото кликабельны)


В центре располагается окно 3д сцены. По умолчанию оно содержит три объекта:
Куб - стандартный объект (примитив)
Лампа - для освещения сцены
Камера - для определения точки отображения сцены.

Старые версии Blender могут запускаться со сценами другого вида, но идея остается той же. 3D-курсор в центре куба используется для определения места добавления новых объектов. Вы можете перемещать его щелчком левой кнопки мыши.

Типы Окон

Blender имеет различные типы окон, и каждое окно может быть переключено в любой тип. Например, ваш экран при запуске программы имеет 5 окон (как на иллюстрации выше), Окно Информации с настройками среды сверху, Окно 3D вида и Окно Шкалы Времени. Справа Окно Структуры Сцены и Окно Свойств Объекта. Кнопка изменения типа окна находится в левом нижнем, либо в левом верхнем углу каждого окна. Пока же, нас интересует только 3д вид.
Изменять размер окна можно потянув за его край левой кнопкой мыши и/или убрав "лишние окна".

Навигация в окне сцены

Менять вид в окне можно несколькими способами: с помощью мыши и с помощью "горячих клавиш". С помошью мыши вращение сцены обеспечивается зажатием колесика мыши и движением. Отдаление/приближение сцены осуществляется вращением колесика. На клавиатуре за вид сцены отвечают клавиши Numpad, а именно:

  1. Numpad 0 - вид из камеры
  2. Numpad 1 - вид спереди
  3. Numpad 2, 4, 6, 8 - поворот
  4. Numpad 3 - вид справа
  5. Numpad 5 - переключение между перспективой и ортогональным видом
  6. Numpad 7 - вид сверху
  7. Numpad 9 - поворот на 180 градусов

Клавиши +/- соответственно приближают и отдаляют сцену, клавиша "точка" фокусирует окно на выделенном объекте.

Простейшие манипуляции с объектами в сцене

Для начала необходимо выделить нужный объект. Это можно сделать, кликнув по нему правой кнопкой мыши, для выделения нескольких объектов нужно зажать клавишу Shift. Чтобы снять выделение нужно нажать клавишу A, и её же, чтобы выбрать все объекты сцены. После того как вы выбрали нужный объкт его граница стала ярко оранжевой, и в центре появлась координатная ось.

Для перемещения объекта следует потянуть за стрелочки левой кнопкой мыши. Эту же операцию можно осуществить и с помощью горячих клавиш, нажав на клавиатуре G, а затем клавишу, соответсвующую букве каждой координатной оси (X, Y, Z). Зажатые лавиши Ctrl и Shift отвечают за "шаг" перемещения. Чтобы применить действие нужно нужно нажать лио левую кнопку мыши либо клавишу Enter.

Аналогично перемещению, объект можновращать, принцип точно такой же, только кроме клавиши G необходимо R.

К простейшим операциям также относится масштабирование. За это отвечает клавиша S. В остальном же, используемые комбинации клавиш аналогичны предыдущим операциям.

Для отмены какого-либо совершенного действия необходимо нажать комбинацию клавишь Ctrl+Z, что восстановаить отмененное действие - Ctrl+Shift+Z

Для того чтобы что то добавить в сцену необходимо нажать Shift+A и выбрать нужный объект, а чтобы удалить, клавишу X или Delete. Чтобы копировать объект нужно нажать Ctrl+C, Ctrl+V для вставки. Либо воспользоваться операцией клонирования Shift+D, сразу же создается полная копия выделенного объекта, и далее можно используя клавиши соответсвующим координатным осям и мышь, переместить объект на нужное место.

Все вышеперечисленные операции также доступны на вкладке Tools (по умолчанию панель слева). Показать/скрыть её можно нажав клавишу T

Также можно скрывать объекты, за это отвечает клавиша H, Alt+H показывает все скрытые объекты

Для выполнения практического задания необходимо смоделировать игрушечный паровозик с помощью примитивов, используя операции, изученные на уроке.
