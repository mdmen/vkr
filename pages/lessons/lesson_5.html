Создание игры


Уникальной особенностью Blender является встроенный в него так называемый "игровой движок" (game engine). Это программный компонент, который позволяет создавать небольшие игры, не выходя из среды трехмерного моделирования Blender.

Blender Game engine:

  1. придает объектам "физические" свойства; объекты начинают вести себя так, как в реальном мире: падают под действием силы тяжести, сталкиваются и т.д. (возможно следует отметить, что в этом смысле движок еще "сыроват");
  2. позволяет наблюдать поведение объектов в реальном времени и воздействовать на них;
  3. записывает происходящее в режиме игры в анимацию;
  4. позволяет визуально настраивать (программировать) поведение и взаимодействие объектов;
  5. а также встраивать скрипты на языке программирования Python.

Создадим простенькую игру на основе ранее созданной сцены. Для начала активируем рендер Blender Game и откроем Logic Editor.
Для описания поведения объектов в Blender используются специальные объекты: сенсоры, контроллеры и актуаторы. Попробуем разобраться в их работе на примере, заставим наш паровозик двигаться:

  1. Выделяем паровозик
  2. На панели Logic Editor в колонке Sensors нажимаем Add sensor -> Keyboard
  3. Кликаем левой кнопкой мыши в поле Key и жмем W
  4. Далее в колонке Controllers нажимаем Add controller -> And
  5. И в колонке Actuators -> Add actuator -> Motion, указываем значение координаты Loc X в 0.01
  6. Соединяем все элементы как показано на фото ниже
  7. Запускаем игру (клавиша P), и теперь при нажатии на W паровозик едет вперед
  8. Проделываем ту же последовательность действий для движения паровозика назад
  9. Далее проделываем все то же самое для клавиш A,D (соответсвенно поворот влево/вправо), но вместо значения Location координаты X в актуаторе motion указываем значение Rotate, и приращение угла

Добавим в сцену немного "физики". Объекты будут иметь массу и взаимодействовать при столкновении:

  1. Выбираем паровозик, идем на вкладку Physics, в поле Physics type выбираем Dynamic
  2. Затем активируем флажок Collision Bounds, выбираем Convex Hull
  3. Для остальных лбъектов в сцене выбираем тип физики Rigid body, collision bounds также Convex hull
  4. Для остальных лбъектов в сцене выбираем тип физики Rigid body, collision bounds также Convex hull

Теперь можно запустить игру и посмотреть что получилось

Создайте игру из существующей сцены, можете добавить произвольных объектов и анимировать их по своему усмотрению, также поэксперементируйте с контроллерами, актуаторами и сенсорами.
